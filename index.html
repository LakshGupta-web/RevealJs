# ETL: Load earnings CSV, compute QoQ growth, push to dashboard
import pandas as pd

# Pretend data
q1 = pd.Series({"revenue": 1146.2, "eps": 0.98})
q2 = pd.Series({"revenue": 1240.5, "eps": 1.06})

qoq = (q2 - q1) / q1
print("QoQ Growth%:", (qoq * 100).round(2).to_dict())

Key Financial Equations

Net Present Value (NPV):

\[ \mathrm{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t} \]

Compound Annual Growth Rate (CAGR):

\[ \mathrm{CAGR} = \left( \frac{\text{Ending}}{\text{Beginning}} \right)^{\tfrac{1}{n}} - 1 \]

Return on Equity (ROE):

\[ \mathrm{ROE} = \frac{\text{Net Income}}{\text{Shareholders'\,Equity}} \]
